以下内容是关于 **Vibe Coding** 的一个提示词模板，旨在帮助没有编程背景的用户生成用于自动合并多种格式的文件（PDF、Word、图片等）的脚本。

## 中文版 (Chinese Version)

**尝试 Vibe Coding:**

下面的提示词适合没有编程背景的同学来尝试 vibe coding，纯大白话，写这样的提示词没有任何难度。
**提示词：**

我没有编程经验，帮我写一个可以直接用的 **Python 脚本**。

我的需求：

1. **我有一个文件夹，里面放着很多资料文件**
可能包括：
* PDF
* Word 文档
* PPT
* Excel
* 图片（jpg、png）
（不一定每种都有）


2. **我希望用一个脚本，把这个文件夹里的所有文件：**
* 自动整理
* 最终合并成一个 **PDF** 文件


3. **合并后的 PDF 要满足：**
* 每一个原始文件，在 PDF 里都有一个明显的标题
* 标题内容就是原始文件名


4. **使用体验要求：**
* 我不懂代码，只能照着步骤操作
* 需要手动配置的内容放到脚本开头（比如文件夹路径）
* 不需要我自己研究技术细节


5. **关于环境和工具：**
* 我只知道需要安装 **Python**
* 其他需要的东西，请你帮我考虑好
* 如果我电脑上没有某些工具，也不要让程序直接报错崩溃
* 能处理的文件就处理，不能处理的就跳过并告诉我
* 输出详细的日志信息，以便于我遇到错误时发送给 AI 进行分析

## English Version (Translation)

**Trying Vibe Coding:**

The following prompt is suitable for students without a programming background to try vibe coding. It's written in plain language, making it very easy to create such prompts.
**Prompt:**

I have no programming experience. Please help me write a **Python script** that I can use directly.

My requirements:

1. **I have a folder containing many source files.**
These may include:
* PDF
* Word documents
* PPT
* Excel
* Images (jpg, png)
(Not all types may be present)


2. **I want a script that takes all files in this folder and:**
* Automatically organizes them
* Finally merges them into a single **PDF** file


3. **The merged PDF must satisfy:**
* Each original file should have a clear heading in the final PDF.
* The heading content should be the original filename.


4. **User Experience Requirements:**
* I don't understand code; I can only follow step-by-step instructions.
* Put parts that require manual configuration at the beginning of the script (e.g., the folder path).
* I don't want to research technical details myself.


5. **Regarding Environment and Tools:**
* I only know that **Python** needs to be installed.
* Please take care of any other necessary dependencies for me.
* If my computer lacks certain tools, do not let the program simply crash with an error.
* Process the files that can be handled, skip those that cannot, and let me know.
* Output detailed log information so that I can send it to an AI for analysis if I encounter errors.
